Brug af fyrværkeri

Fyrværkeri må bruges fra 27. december til og med 1. januar. Sikkerhedsstyrelsen er myndighed på fyrværkeriområdet. Fyrværkeri er som udgangspunkt farligt, og derfor er det vigtigt at udvise en sikker adfærd. Blandt andet ved at følge sikkerhedsinstruktionen på produktet.
