Tændskabe til gadebelysningsanlæg (April 2000)

Tændskabe til gadebelysningsanlæg i større byer har elselskaberne traditionelt selv lavet, og elselskaberne har sædvanligvis også vedligeholdt gadebelysningsanlæggene.


Artiklen fra: Elektricitetsrådet
Har tidligere været bragt i Kabel- og liniemesteren april 2000

Andre steder har lokale elinstallatører stået for opgaven.

Eksisterende tændskabe
Eksisterende tændskabe kan bibeholdes, så længe kapslingen er i orden og det elektriske materiel ikke er beskadiget eller slidt op. Krav om beskyttelse mod indirekte berøring (ekstrabeskyttelse) blev først indført den 1. april 1994, da Stærkstrømsbekendtgørelsen, Elektriske installationer (SBEi) afløste det lysegrønne Stærkstrømsreglement.

Nye tændskabe efter EN 60439 eller efter ELRÅD-MEDDELELSE, Materiel nr. 3/99
Nye tændskabe kan udføres efter de nye europæiske standarder for Lavspændingstavler, Stærkstrømsbekendtgørelsens Afsnit 439-1 til 439-5. Mindre, nye tændskabe kan også laves efter ELRÅD-meddelelse, Materiel nr. 3/99, hvori der bl.a. står, hvorledes sammenbygninger, som ikke er omfattet af tavlestandarderne, kan fremstilles - herunder styretavler.

Sådanne styretavler, som kun indeholder måle- og indikeringsudstyr, betjeningsknapper, kontaktorer, relæer m.v. samt eventuelt overstrømsbeskyttelsesudstyr og udstyr til beskyttelse mod elektrisk chok, der funktionsmæssigt hører sammen med det øvrige udstyr, behøver ikke at følge 439-serien.

Tændskabe med et par styrede afgange med kontaktorer, der aktiveres af skumringsrelæer eller ure, kan betragtes som styretavler, der ikke behøver at følge tavlestandarden.

Der er for tiden ingen speciel standard, der dækker sådanne styretavler, så i stedet henvises til SBEi med hensyn til bl.a. beskyttelse mod direkte og indirekte berøring, kapslingsklasse m.v. Et dobbeltisoleret skab af passende størrelse i kapslingsklasse IP34 med værktøjslukke i lågen vil være anvendeligt til sådanne mindre tændskabe, idet der for det indbyggede materiel skal iagttages eventuelle fabrikantanvisninger om temperatur, kortslutningsholdbarhed o.l..

Eftersyn af gadebelysningsanlæg
Elektricitetsrådet skrev i juli 1997 til landets amter, kommuner, elselskaber og boligselskaber om, hvordan elulykker ved udendørs lamper kan undgås og understregede betydningen af regelmæssige eftersyn.

Vedlagt brevet var en anbefaling af, hvordan sikkerhedsmæssig kontrol af udendørs lysanlæg på offentlige steder kan gribes an. Interesserede kan få anbefalingen ved henvendelse til Rådets kontor.
