Projektgodkendelse for biogasanlæg

Følg de ni vejledninger, så sagsbehandlingen går så hurtigt som muligt.


Der er nye regler om gassikkerhed fra 21. april 2018.

Læs mere om de nye regler på   www.gassikkerhedsregler.dk   

Ved at følge vejledningerne nedenfor lettes sagsbehandlingen i Sikkerhedsstyrelsen. Det er vigtigt at følge vejledningerne og være opmærksom på at få vedhæftet alle relevante dokumenter sammen med ansøgningen, så ventetiden bliver så kort som muligt. Styrelsen arbejder på at digitalisere ansøgningerne, så godkendelsesprocessen bliver enklere.

Hent de ni vejledninger her:

1. Projektgodkendelse for biogasanlæg uden gasforbrugende apparater eller ombygning af eksisterende biogasanlæg (PDF)
2. Projektgodkendelse for biogasanlæg med gasforbrugende apparater (PDF)
3. Projektgodkendelse for biogasanlæg med gasforbrugende apparater og én forbruger (PDF)
4. Projektgodkendelse for biogasanlæg med gasforbrugende apparater og opgraderingsanlæg (PDF)
5. Projektgodkendelse for udskiftning af gasforbrugende apparater på eksisterende biogasanlæg og evt. etablering af afsvovlingsanlæg (PDF)
6. Projektgodkendelse for PE-distributionsnet til biogas, hvor methanindholdet er ≤ 80% (PDF)
7. Vejledning til lægning og tæthedsprøvning af PE-ledninger til biogas (PDF)
8. Vejledning vedrørende PE-gasrørsindføring i bygninger og brønde samt PE-ledninger ved rådnetanke, gaslager, gasfakkel og udendørs behandlin... (PDF)
9. Vejledning vedrørende drifts- og vedligeholdelsesplan for biogasanlæg med gasforbrugende apparater (PDF)
