Suttekæder kan være farlige

Sikkerhedsstyrelsen har testet 28 suttekæder, som er solgt på det danske marked. Kun én levede op til sikkerhedskravene i standarden.

Suttekæder kan se både søde og nuttede ud og findes i et utal af variationer, men langt de fleste, som Sikkerhedsstyrelsen har testet, lever altså ikke op til de gældende sikkerhedskrav i standarden.

- Det mest almindelige problem er, at kæden til sutten er for lang. Det betyder, at den kan vikle sig rundt om barnets hals, og i værste fald risikerer barnet at blive kvalt, fortæller Dorthe G. Rasmussen, som er pressechef i Sikkerhedsstyrelsen. Den korrekte længde i henhold til standarden er max 22 cm.

Et andet typisk problem er, at kæderne er pyntet med farvestrålende små dele, som også kan udgøre en kvælningsrisiko for børnene.

”Suttekæder findes jo i mange forskellige designs, og resultaterne af testene indikerer, at man skal være kritisk i forbindelse med dette produkt. En suttekæde, uanset hvor nuttet den ser ud, er jo ikke streng nødvendig for, at barnet kan bruge sin sut. Man kan som alternativ overveje at købe nogle ekstra sutter til børnene i stedet, siger Dorthe G. Rasmussen.

Sikkerhedsstyrelsen vil nu henlede Dansk Erhvervs opmærksomhed på, at importører og detailhandel tilsyneladende ikke er tilstrækkelige opmærksomme på, at der findes en standard med sikkerhedskrav for suttekæder.

De fleste af de produkter, som er fejlet i Sikkerhedsstyrelsens test, er fjernet fra markedet og information om produkterne lægges løbende på styrelsens hjemmeside, efterhånden som sagsbehandlingen afsluttes.
