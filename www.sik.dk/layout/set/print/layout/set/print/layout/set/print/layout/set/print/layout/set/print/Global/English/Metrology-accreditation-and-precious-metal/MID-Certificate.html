MID Certificate

A MID certificate is an EU type examination certificate issued by a notified bodies in accordance with module B or H1 in the Measuring Instruments Directive (MID), or a certificate of a manufacturer's approved quality management system in accordance with module D. Here you can find certificates issued by Danish notified bodies.


You can search in the database of MID certificates by using the search function below. The database contains information about the applicant's name, address and instrument types.

Certificates can be obtained in PDF format.

Search guidelines
A complete list of the database's content is available by clicking on "search".

Searching can be limited
In the uppermost search field, you can search for names, addresses, numbers, instrument types, etc.
If, for example, you want certificates from a specific company , you type the company's name in the search field and then click on "search". The search result will then only contain information about the searched company.

In the bottommost search field, the search can be limited to a specific measurement category or approved quality systems.

List of certificates in a specific measurement category
put a check mark in the category wanted and click on ”search". The search result will then only contain information about the category wanted.

List of manufacturers who have an approved quality management system
put a check mark by ”Approved quality management system” and click on "search". The search result will then only contain information about the manufacturers with approved quality management systems.

Searching can be combined
Information and certificates from a specific manufacturer, which has an approved quality management system within a specific measurement category:
type the name of the manufacturer in the uppermost search field and put a check mark by both the measurement category wanted and the approved quality management systems.
