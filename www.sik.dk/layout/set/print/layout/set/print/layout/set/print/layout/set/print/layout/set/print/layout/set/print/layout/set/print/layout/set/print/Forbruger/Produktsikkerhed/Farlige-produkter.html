Farlige produkter

Vi offentliggør farlige produkter, som vi finder på det danske marked. Hent billede af produktet og få nærmere oplysninger om fejlen ved produktet.

Register over farlige produkter

Når vi får kendskab til et produkt, som muligvis ikke overholder kravene, bliver det undersøgt nærmere. Viser det sig at være farligt, bliver det offentliggjort her på hjemmesiden.

Farlige produkter fundet i EU

Når et EU-land finder et farligt produkt på markedet, bliver andre EU-lande advaret om produktet. Det sker igennem det europæiske system for hurtig varsling, også kaldet RAPEX.

Produkter til børn og babyer

Find produkter til børn og babyer, som er vurderet farlige af Sikkerhedsstyrelsen. Se foto og beskrivelse af de farlige produkter. Læs også om, hvordan du skal forholde dig, hvis du ejer et af de farlige produkter.

Elprodukter

Se foto og beskrivelse af elprodukter, som Sikkerhedsstyrelsen har undersøgt og klassificeret som farlige. Læs også om, hvordan du skal forholde dig, hvis du ejer et af de farlige elprodukter.

Gasprodukter

Her finder du beskrivelse og foto af farlige gasprodukter. Læs også om, hvordan du skal forholde dig, hvis du ejer et af de farlige gasprodukter.

Fyrværkeri

Her kan du finde en række fyrværkeriartikler, som er blevet testet farlige. Hvis du har købt fyrværkeri, der er ulovligt, eller som er blevet trukket tilbage fra markedet, anbefaler Sikkerhedsstyrelsen, at du returnerer det til forhandleren eller bortskaffer det som farligt affald.

Andre produkter

Andre farlige produkter, som vi har dømt farlige, er samlet her. Du kan finde beskrivelse og foto af de farlige produkter og læse om, hvordan du skal forholde dig, hvis du har et af produkterne.

Arkiv over farlige produkter

Når et farligt produkt er blevet tilbagekaldt eller tilbagetrukket, afslutter vi sagen. Du kan herefter finde produktet i vores arkiv. Arkivet er oprettet med henblik på forbrugerinformation.

Andre advarsler på forskellige produkter

Sikkerhedsstyrelsen holder dig orienteret, når forhandlere offentliggører tilbagetrækkelser af produkter. Det er også her, du kan finde særlige advarsler i forbindelse med et produkt.

Anmeld et farligt produkt

Hvis du har mistanke om, at et produkt er farligt, kan du altid henvende dig til Sikkerhedsstyrelsen. Vi vil så afgøre, om produktet er farligt. Det kan nemlig også være, at produktet er slidt, eller at det bliver brugt forkert.
