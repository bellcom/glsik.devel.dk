Flaskegas i festtelte/havepavilloner

Se, hvilke regler der gælder for brug af terrassevarmere og gasgrill i festtelte og havepavilloner


Få flere gode råd til sikker brug af flaskegas i folderen "5 gode råd før du gi´r den gas".

Link til folder

Terassevarmere
Du må kun bruge terrassevarmere i festtelte og havepavilloner, hvis minimum 25 procent af vægarealet er åbent. Det er for at sikre den nødvendige ventilation. Det er farligt, hvis ikke der kommer den nødvendige mængde luft ind.

Gasgrill
Reglerne for at bruge en gasgrill er strengere. Her skal minimum halvdelen af vægarealet være åbent.
