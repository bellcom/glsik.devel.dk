Markedsovervågning af nye områder

Sikkerhedsstyrelsen overtager i løbet af 2018 opgaver med markedsovervågning for en række andre myndigheder. Det er led i regeringens plan for flytning af offentlige arbejdspladser, Bedre balance II.

De første områder overtages af Sikkerhedsstyrelsen 1. august 2018. Se mere om dem her.

EMC og RED

Fra 1. august 2018 udfører Sikkerhedsstyrelsen markedsovervågning for radioudstyr samt elektromagnetiske forhold ved produkter på vegne af Energistyrelsen.

ECO-design og energimærkning

Sikkerhedsstyrelsen skal fra 1. august 2018 udføre markedsovervågning med eco-design og energimærkning af elprodukter for Energistyrelsen. Det praktiske arbejde med kontrollen udføres af Sekretariatet for Ecodesign og Energimærkning, som hidtil har løst opgaven for Energistyrelsen.

Eksplosivstoffer til civil brug

Sikkerhedsstyrelsen fører fra 1. august 2018 kontrol med eksplosivstoffer til civil brug på vegne af Justitsministeriet.

Fritidsfartøjer

Sikkerhedsstyrelsen markedsovervåger fra 1. oktober 2018 fritidsfartøjer på vegne af Søfartsstyrelsen.
Aftalen gælder fritidsfartøjer, der er omfattet af fritidsfartøjsdirektivet (EU-direktiv 2013/53/EU).
