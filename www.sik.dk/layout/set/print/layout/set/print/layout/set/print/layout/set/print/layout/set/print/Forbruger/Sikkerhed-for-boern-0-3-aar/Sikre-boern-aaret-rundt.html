Sikre børn året rundt

Hold styr på sikkerheden både udendørs og indendørs og både sommer og vinter. Find de gode råd til årstider og højtider.

Fastelavn

Glæder dit barn sig også til fastelavn? Det forstår vi godt, for det er sjovt at klæde sig ud. Vi har samlet en række sikre råd om fastelavnskostumet.

Forår, sommer og udeliv

Når det er sæson for trampoliner, gynger, boldspil og grill, skal udelivet være sikkert.

Sikker påske

Påsken er en hyggelig tid, og måske kommer påskeharen også på besøg hos jer med små, lækre påskeæg og andre godter. Men pas på med påskeæg, småt legetøj og andet, der kan puttes i munden.

Efterår og vinter

Årets kolde måneder kræver enten varmt tøj eller indesysler. Hold øje med det, der kan være farligt for de små børn.

Pas på de små til halloween

Sørg for, at børnene ikke kommer til skade med de uhyggelige kostumer. Få her gode råd til halloween-kostumet.

Jul uden uheld

Julen er i høj grad børnenes tid, og i mange børnefamilier kommer ”nissen” forbi med små kalendergaver og lækkerier, som kommer i julesokken. Hvad nissen måske ikke tænker over er, at gaven måske indeholder små dele, som kan være farligt for børnene.

Nytår uden skader

Følg fyrværkerirådene og hav respekt for fyrværkeriet.
