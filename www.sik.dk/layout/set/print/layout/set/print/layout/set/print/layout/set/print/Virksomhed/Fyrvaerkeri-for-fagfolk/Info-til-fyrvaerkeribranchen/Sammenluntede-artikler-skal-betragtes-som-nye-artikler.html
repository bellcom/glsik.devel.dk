Sammenluntede artikler skal betragtes som nye artikler

Læs, hvad du skal være opmærksom på, hvis du sammenlunter 2 eller flere fyrværkeriartikler.


Sikkerhedsstyrelsens holdning til alle typer af sammenluntede fyrværkeriartikler er:

  • Hvis 2 eller flere artikler bliver sammenluntet, er der tale om en ny artikel
  • Det kræver en ny godkendelse af transportklassifikationen
  • Der skal indsendes en serie 6 test, ellers kan artiklen kun godkendes som 1.1G.

Det gælder uanset om der er tale om konsumfyrværkeri eller professionelt fyrværkeri.

Vær desuden opmærksom på, at hvis sammenluntede fyrværkeriartikler ønskes bragt i omsætning, kan det kræve en ny overensstemmelsesvurdering.
