Bor du sikkert, når det handler om el?

Elektricitetsrådet slår nu et slag for, at unge får indrettet sig sikkert, når de flytter hjemmefra. I hvert fald sikkert med elektricitet.


Pressemeddelelse
2. september 2002

Statistikken over elbrande og elulykker viser, at der er en stigning i elulykker og elbrande, hvor unge i aldersgruppen fra 16-30 år er indblandet. Hvert år kommer omkring 350 unge til skade på grund af elektricitet. Mange ulykker og brande kunne være undgået, hvis vi respekterede og huskede den fare, elektriciteten rummer.

Elsikkerhed handler om at være på den sikre side, når det drejer sig om brugen af elektricitet. Det handler om at komme ulykkerne i forkøbet.

Elektricitetsrådet har produceret et informationskort, der giver fire enkle og nyttige råd:

  • Sørg for, at der er en HFI/HPFI-afbryder der, hvor du bor. Den sidder i eltavlen og afbryder strømmen, hvis der sker fejl.
  • Test afbryderen et par gange om året ved at trykke på den lille prøveknap. Så ved du, at den fungerer.
  • Hold orden i dine ledninger. Undgå for mange løse ledninger og stikdåse på stikdåse. Ledninger skal ligge, så isolationen ikke ødelægges. Ødelagt isolation giver strømmen frit spil.
  • Lav ikke selv elektrisk arbejde, uden at du på forhånd har undersøgt, hvad du selv må lave, og hvordan du gør det forsvarligt. Hjemmelavede og halvfærdige elektriske konstruktioner kan koste liv.

Informationskortet distribueres i samarbejde med blandt andet bolighuse og kollegieadministrationer. Derudover vil der være gratis postkort på caféer og på uddannelsessteder. Med collagen af avisudklip skal kortene prikke til de unges bevidsthed om sikkerhed og elektricitet.
