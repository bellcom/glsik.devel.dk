Suttekæder kan være farlige

Sikkerhedsstyrelsen afslører fejlende fyrværkeri

13 ud af 23 testede fyrværkeriartikler har alvorlige funktionsfejl, viser en stikprøve Sikkerhedsstyrelsen har udført - med særligt fokus på formodet farlige fyrværkeriartikler. Fyrværkeriet kan udgøre en forøget risiko for forbrugerne, og der er derfor nedlagt salgsforbud mod de berørte produkter.

Gas fra Tyskland kan give driftsproblemer

Naturgas fra Tyskland har siden oktober været en del af den danske gasforsyning. Gassen giver desværre en række driftsproblemer hos et mindre antal forbrugere. Derfor har Sikkerhedsstyrelsen nu udsendt nye retningslinjer til alle danske gasservicefirmaer.

Test af HPFI-afbryder kan redde dit liv

Er din HFI- eller HPFI-afbryder mere end 10 år gammel, er der risiko for, at den ikke fungerer. Derfor er det vigtigt at teste afbryderen to gange om året, da den kan redde dit liv.

Sikkerhedsstyrelsen har oprustet sin håndtering af overtrædelsessager, og nu viser resultaterne sig.

Bedre produktsikkerhed til gavn for forbrugerne og de erhvervsdrivende

Forbrugerne skal have tillid til de produkter, der omgiver dem. Derfor lægger den nye lov om produktsikkerhed op til, at reglerne fremover bliver nemmere at forstå og overholde for producenter og distributører.
