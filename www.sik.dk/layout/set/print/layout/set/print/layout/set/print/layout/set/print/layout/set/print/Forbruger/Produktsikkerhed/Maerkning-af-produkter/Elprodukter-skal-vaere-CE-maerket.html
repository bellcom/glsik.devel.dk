Elprodukter skal være CE-mærket

CE-mærket er det væsentligste mærke på elprodukter. Med CE-mærket tilkendegiver fabrikanten over for myndighederne, at produktet opfylder kravene til elsikkerhed. Krav der er fælles i hele EU.


Det er vigtigt at være opmærksom, at et CE-mærke ikke behøver at betyde at fabrikanten har betragtet el-produktet som værende et legetøj. CE-mærket kan være påsat alene for el-delen om at gøre. Fx på en lampe er CE-mærke ikke ensbetydende med at der er tale om et sikkert legetøj. 

Inden produktet kan CE-mærkes, skal fabrikanten udfylde en EF-overensstemmelseserklæring. Med denne erklæringen tilkendegiver fabrikanten, hvordan produktet er sikkert, og at det ikke er farligt at anvende.

Manglende mærkning udløser salgsforbud
Det har været obligatorisk for fabrikanter at CE-mærke elektriske produkter siden januar 1997, og Sikkerhedsstyrelsen har gradvist indført en strammere kurs ved kontrol af CE-mærkning.

Sikkerhedsmæssige og tekniske fejl ved produkterne vil automatisk, og som minimum, udløse et salgsforbud. Visse formelle fejl, fx manglende mærkning af produktet, vil også udløse et salgsforbud.

Letter eksport og import
CE-mærket skal gøre det nemmere at eksportere og importere varer inden for EU. Reglerne er derfor ens i alle landene, og de nationale myndigheder kan ikke stille strengere krav til CE-mærkede produkters udformning, end EU's medlemslande er blevet enige om.
