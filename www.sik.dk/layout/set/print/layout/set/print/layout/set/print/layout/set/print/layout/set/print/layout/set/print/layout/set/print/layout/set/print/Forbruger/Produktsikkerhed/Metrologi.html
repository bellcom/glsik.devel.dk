Nye regler for varmefordelingsmålere per 1. marts 2015

Reglerne for varmefordelingsmålere er ændret. Hvis din varmefordelingsmåler er sat op efter den 1. august 1998, skal din varmeleverandør stille en række oplysninger til rådighed for dig som varmeforbruger. Oplysningerne kan du bruge til at vurdere, om grundlaget for den afregning, der sker gennem varmefordelingsmålere, er korrekt.
