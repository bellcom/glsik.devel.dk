Vejledninger om flaskegas

Der er en række ting, som er vigtigt at vide, hvis du bruger flaskegas. Læs fx om placering af gasflasker og flaskegas for campister/friluftsfolk.

Camping-TV

Campingsæsonen er skudt i gang og det er tid til at klargøre campingvognen. Få gode råd til gassikker camping her.

Hvor må gasflaskerne stå?

Se, hvor du må placere gasflasker til gaskomfurer, gaspejse, gaskøleskabe, gasvarmeovne og terrassevarmere.

Hvor mange gasflasker må man have?

Der findes regler for, hvor mange gasflasker man må have derhjemme. Se, hvor mange gasflasker der må være i husholdningen, og læs om hvilke foranstaltninger, du skal være særlig opmærksom på.

Flaskegas i festtelte/havepavilloner

Se, hvilke regler der gælder for brug af terrassevarmere og gasgrill i festtelte og havepavilloner

Flaskegas for campister og friluftsfolk

Der gælder særlige regler for flaskegasinstallationer i campingvogne, autocampere, arbejdsskure, skurvogne og lignende. Se, hvor mange gasflasker du må have i din campingvogn, regler for transport af gasflasker samt forholdsregler ved køb og salg af campingvogne.

Tjek gasslangens tæthed

Som udgangspunkt skal alt arbejde udføres af en autoriseret vvs-installatørvirksomhed, men du må gerne selv tjekke, om gasslangeforbindelsen er tæt.
