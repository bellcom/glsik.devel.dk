Pressefoto og logo

Her finder du pressefotos af Sikkerhedsstyrelsens ledelse og presseteam. Du kan også hente Styrelsens logo.


Pressefoto

Pressebilleder af Sikkerhedsstyrelsens direktør, chefgruppe og presseteam er kun til redaktionelt brug. Klik på billedet for at downloade i stort format.

Logo

Sikkerhedsstyrelsens logo i udvalgte formater og farver. Logoet er kun til redaktionelt brug efter aftale med Sikkerhedsstyrelsen og må ikke bearbejdes, redigeres eller lignende.

Brug af Sikkerhedsstyrelsens logo

Er din virksomhed autoriseret eller godkendt af Sikkerhedsstyrelsen, må du bruge Sikkerhedsstyrelsens logo på fx hjemmeside eller brevpapir, evt. sammen med dit autorisations- eller godkendelsesnummer.
