Register over farlige produkter

Når vi får kendskab til et produkt, som muligvis ikke overholder kravene, bliver det undersøgt nærmere. Viser det sig at være farligt, bliver det offentliggjort her på hjemmesiden.


Vi offentliggør løbende farlige produkter på www.sik.dk .

I registret kan du se beskrivelse og foto af farlige produkter. Vi informerer også om, hvordan du skal forholde dig, hvis du ejer et produkt, som vi har kategoriseret som farligt.

Du kan også selv anmelde et produkt til Sikkerhedsstyrelsen, du mener er farligt.
