Værktøj til elarbejde

Der er flere redskaber, som er vigtige, når du skal lave elarbejde. Se en række af de mest brugte værktøjer, som du bør have i din værktøjskasse.


 

Spændingstester

Sæt af 3-4 isolerede skruetrækkere

 

Bidetang

 

Afisoleringstang eller kniv

Sæt dig grundigt ind i det elarbejde, som du vil udføre og hav det nødvendige værktøj og materiel parat.

Typiske årsager til brand og stød kan være:

  • Løse forbindelser
  • Uisolerede dele i en elinstallation
  • Brug af en forkert type ledning
  • Defekt elmateriel, fx stikprop
  • En stikprop der ikke er samlet korrekt
  • En ledning som er for langt afisoleret
  • Manglende dæksel på elmateriel
  • En skrue eller et søm som går gennem ledninger
  • Ledninger som er forbundet forkert
  • Kobbertråde der klippes over ved afisolering.
