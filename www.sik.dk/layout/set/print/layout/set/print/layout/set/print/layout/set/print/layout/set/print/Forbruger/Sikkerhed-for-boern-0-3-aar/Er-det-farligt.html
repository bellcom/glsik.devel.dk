Er det farligt

Med sikre rammer kan det lille barn bevæge sig trygt omkring. Her kan du læse mere om, hvad du skal være opmærksom på, når det gælder produkter, og hvordan du kan undgå farlige situationer.

Test sikkerheden med en legetøjstester

Børn kan blive kvalt i små dele fra legetøj. En legetøjstester hjælper dig med at teste, om legetøj eller smådele i dit hjem er så små, at de er farlige for dit barn i alderen 0-3 år.

Print din egen legetøjstester

Du kan printe og klippe din helt egen legetøjstester, som kan hjælpe dig med at teste, om smådelene i dit hjem er så små, at børn i alderen 0-3 år risikerer at blive kvalt. Se her, hvordan du samler og anvender den.

Test din børnesikkerhed

Prøv spillet 'Forældrejagten', og test hvor børnsikker du er. Her skal du finde alle de farlige ting, der kan være til skade for små børn i stuen og køkkenet.

Liste over farlige produkter til børn og babyer

Får vi kendskab til et produkt, som kan være farligt for børn, undersøger vi det. Viser det sig, at produktet er farligt, offentliggør vi det på en liste over farlige produkter til børn og babyer.

Farlige produkter fundet i EU

Når et EU-land finder et farligt produkt på markedet, bliver andre EU-lande advaret om produktet. Det sker igennem det europæiske system for hurtig varsling, også kaldet RAPEX.
