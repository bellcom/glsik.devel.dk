Info til kommuner om fyrværkeri

Her kan du finde relevant information til kommunerne og vejledning om opbevaring og anvendelse af professionelt fyrværkeri.

Fyrværkerkurser for beredskaberne

Det er muligt for beredskaberne at deltage på fyrværkerkurser. Man aflægger ikke prøve, og modtager ikke kursusbevis eller godkendelse som festfyrværker, men deltager til reduceret pris.

Salg og opbevaring af fyrværkeri

Se, hvilke tilladelser henholdsvis Sikkerhedsstyrelsen og kommunerne giver ved fyrværkeri. Hent også link til spørgsmål og svar om salg og opbevaring af fyrværkeri.

Opbevaring og håndtering af fyrværkeriaffald på genbrugspladser

Vi vil i forbindelse med det kommende nytår gerne oplyse om, hvordan opbevaring af konsumfyrværkeriaffald på genbrugspladserne kan håndteres.

Håndtering af ansøgninger om professionelt fyrværkeri

Vi har lavet nogle vejledninger til kommunerne, når de skal behandle ansøgere, som søger om godkendelser i forbindelse med professionelt fyrværkeri.

Temadage for de kommunale beredskaber

Hvert år afholder vi en temadag for de kommunale beredskaber, hvor vi informerer om reglerne for anvendelse, salg og opbevaring af fyrværkeri.
