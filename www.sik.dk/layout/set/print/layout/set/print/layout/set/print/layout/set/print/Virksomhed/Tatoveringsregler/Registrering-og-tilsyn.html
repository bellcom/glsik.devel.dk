Registrering og tilsyn

Sikkerhedsstyrelsen vil gennemføre tilsyn med alle registrerede tatoveringssteder.

Registrer dit tatoveringssted eller ændr oplysninger

Tatoveringssteder skal registrere sig via en blanket på Virk.dk.

Registreringskrav for tatoveringssteder

Alle tatoveringssteder i Danmark skal registreres. Det følger af tatoveringsloven, der er trådt i kraft 1. juli 2018.

Register over tatoveringssteder

Her finder du en oversigt over tatoveringssteder, der er registreret hos Sikkerhedsstyrelsen.

Tilsyn med tatoveringssteder

Sikkerhedsstyrelsen vil gennemføre tilsyn med alle registrerede tatoveringssteder. I 2019 kan et tatoveringssted vælge kun at få et vejledningsbesøg.

Hvis tatoveringsloven bliver overtrådt

Overtrædelser af tatoveringsloven kan straffes med bøder.
