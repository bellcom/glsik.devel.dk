Del af Erhvervsministeriets koncern

Sikkerhedsstyrelsen er en styrelse i Erhvervsministeriet og udgør sammen med ministeriets departement og øvrige 6 styrelser og 8 institutioner Erhvervsministeriets koncern.


Styrelsernes direktører er medlemmer af koncernledelsen, og koncernen ledes af departementschefen.
 
Koncernen arbejder ud fra et fælles overordnet grundlag bestående af mission og vision samt et værdi- og ledelsesgrundlag.
 
Det er koncernledelsens opgave at rådgive departementet og træffe beslutninger i overordnede strategiske spørgsmål om gennemførelse af regeringens politik.
 
Hvert år formuleres en mål- og resultatplan mellem styrelserne og departementet med henblik på at måle på styrelsernes indsats.
