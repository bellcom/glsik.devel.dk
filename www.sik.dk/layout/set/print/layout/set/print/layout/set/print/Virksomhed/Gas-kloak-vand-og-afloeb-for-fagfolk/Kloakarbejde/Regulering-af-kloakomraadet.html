Regulering af kloakområdet

Sikkerhedsstyrelsen er myndighed på den del af kloakområdet, som handler om autorisation. Det egentlige kloakarbejde reguleres igennem bygningsreglementet, der hører under Energistyrelsen.


Gå ind på bygningsreglementet og læs mere om reglerne for kloakarbejde.

Link Bygningsreglementet.dk

Hvis du vil søge om autorisation som kloakmestervirksomhed, kan du læse mere om autorisation og gældende autorisationslovgivning på Sik.dk.
