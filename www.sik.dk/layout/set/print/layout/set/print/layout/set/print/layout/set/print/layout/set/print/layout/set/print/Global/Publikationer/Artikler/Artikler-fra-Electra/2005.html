Strømkredse skal kunne adskilles (Februar 2005)

Alle strømkredse skal kunne adskilles. Men når man skal vælge adskiller, skal der tages hensyn til, om adskilleren skal betjenes af lægmand, og i hvilken forbindelse adskilleren skal anvendes.

Den ildesete gruppetavle (August 2005)

Gruppetavler skal anbringes, så betjening, eftersyn, vedligeholdelse og adgang til ledningsforbindelser kan foregå så let som muligt. Disse muligheder må ikke forringes i væsentlig grad ved anbringelse af gruppetavlerne i skabe eller lignende

Mere samarbejde om kontrollen af nye elektriske installationer (April 2005)

I 2004 blev der foretaget tilsyn med ca. 11 procent af de 34.748 færdigmeldte installationer, som opfyldte de kriterier, der ligger til grund for udvælgelse til tilsyn. Det blev året, hvor der blev fundet færrest fejl per gennemført tilsyn

Fyrværkerilagre og elektriske installationer (December 2005)

Fyrværkeri kan være farligt. Derfor kan der være skærpede krav til den elektriske installation på de steder, hvor man producerer og opbevarer fyrværkeri.

Nye og forbedrede SKS-retningslinier (Oktober 2005)

Den 1. januar 2006 skal alle virksomheder tage aktivt stilling til alle SKS-kravene

Installationer i eksplosionsfarlige områder (Juni 2005)

Eksplosionsfarlige områder skal klassificeres i zoner. En omhyggelig klassificering i korrekte zoner er væsentlig for såvel elsikkerhed som økonomi. Det går ikke længere at strø om sig med unødvendigt strenge zoner, da det vil medføre kostbare sikkerhedskrav til materiellet
