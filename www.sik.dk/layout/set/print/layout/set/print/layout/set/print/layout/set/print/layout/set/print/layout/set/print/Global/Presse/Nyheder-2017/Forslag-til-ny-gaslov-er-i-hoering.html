Forslag til ny gaslov er i høring

Det er muligt at kommentere på lovforslaget frem til 6. oktober 2017. Gassikkerhedsloven forventes at træde i kraft i foråret 2018. 


Lovforslaget er en modernisering og forenkling af den nuværende lov om gasinstallationer og gasmateriel. Loven vil blive suppleret af en række bekendtgørelser, der erstatter det gældende gasreglement. Lovforslaget tilpasser desuden de danske regler til en ny gasapparatforordning i EU. Den træder i kraft 21. april 2018.

Find lovforslaget og et svarskema til høringssvar på Høringsportalen.
