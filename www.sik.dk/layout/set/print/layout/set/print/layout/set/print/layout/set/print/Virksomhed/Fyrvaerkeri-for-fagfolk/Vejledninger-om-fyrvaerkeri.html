Vejledning om salg, håndtering og opbevaring af fyrværkeri

Hent vejledning til de tekniske forskrifter for salg, håndtering og opbevaring af fyrværkeri. Vejledningen er et bilag til bekendtgørelsen om fyrværkeri.

Fyrværkeri på fodboldstadions kan være farligt

Fyrværkeri på fodboldstadions er udbredt. Men fyrværkeri kan være farligt på steder med trange pladsforhold og mange mennesker. Folderen henvender sig til både politifolk ved fodboldkampe og til kontrollører, som arbejder på fodboldstadions.
