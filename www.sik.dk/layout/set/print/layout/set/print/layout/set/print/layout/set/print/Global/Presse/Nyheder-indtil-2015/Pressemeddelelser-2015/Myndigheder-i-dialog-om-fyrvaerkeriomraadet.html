Myndigheder i dialog om fyrværkeriområdet

Med baggrund i ulykkerne omkring nytåret 2014-2015 med luftbomber afholder Sikkerhedsstyrelsen, SKAT og politiet et dialogmøde.


Formålet med mødet er i samarbejde mellem myndighederne at kortlægge mulighederne for yderligere tværgående kontrol og initiativer til at dæmme op for ulovlig indførelse, erhvervelse, opbevaring og anvendelse af luftbomber og lignende fyrværkeriartikler.

Dialogmødet vil få deltagelse fra politiet, de kommunale beredskaber, SKAT og Sikkerhedsstyrelsen, og derudover ventes bl.a. repræsentanter for fyrværkeribranchen at blive inviteret. Med en bred deltagerkreds vil der være mulighed for at vurdere alle tilgængelige værktøjer mht., om de er tilstrækkelige eller kan udbygges.

Der er endnu ikke fastsat en dato for dialogmødet, men målsætningen er, at det skal foregå i løbet af januar.
