Få styr på solcellerne

Tjek sikkerheden på dit solcelleanlæg. Der kan være en sikkerhedsrisiko ved nogle solcelleinstallationer, fordi der ikke er installeret den type fejlstrømsafbryder, som kræves til inverteren.


Derfor skal du tjekke, hvilken type fejlstrømsafbryder du har, og hvad installationsvejledningen til inverteren foreskriver. En fejlstrømsafbryder type B vil altid være dækkende.

Se pressemeddelelse af 21. november 2013 fra Sikkerhedsstyrelsen

I vejledningen til solcellejere pkt. 9 (se link nederst på siden) finder du hjælp til at tjekke, hvilken type fejlstrømsafbryder, du har.

Overblik over din solcelleinstallation
I løbet af de seneste to år er antallet af solcelleanlæg i Danmark vokset fra ganske få til mere end 87.000 anlæg.

Sikkerhedsstyrelsen har udarbejdet en vejledning, så ejere af solcelleanlæg kan tjekke, at deres solcelleinstallation er korrekt udført.

Vejledningen er bygget op som 12 punkter. De svarer til emnerne i en vejledning til autoriserede elinstallatørvirksomheder og andre, som arbejder professionelt med opsætning og vedligeholdelse af solcelleinstallationer.

Hvis du efter at have gennemgået tjeklisten er i tvivl, om din installation er udført korrekt, bør du kontakte en autoriseret elinstallatørvirksomhed eller leverandøren af solcelleanlægget.
