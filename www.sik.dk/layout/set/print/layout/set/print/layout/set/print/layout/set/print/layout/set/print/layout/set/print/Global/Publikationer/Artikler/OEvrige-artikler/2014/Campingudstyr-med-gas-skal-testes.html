Campingudstyr med gas skal testes

Sikkerhedsstyrelsen vil i løbet af 2014 sætte fokus på campingudstyr, som bruges med små engangs-gasdåser. Nogle af de mange typer kogeblus, campinglygter, små grill og andre produkter skal kontrolleres og testes, og formålet er at få et overblik over sikkerheden ved campingprodukterne.


Meget campingudstyr er beregnet til små gasdåser, fordi det gør udstyret kompakt og let at medbringe. Det er planlagt, at forskellige typer udstyr skal udtages til markedskontrol og test. Det skal kontrolleres, om udstyret er mærket korrekt og har den tekniske dokumentation og den vejledning, der kræves. Eksempelvis skal brugsvejledningen være udformet på dansk, hvis udstyret sælges i Danmark. Derudover sendes eksemplarer af udstyret til funktionstest på et akkrediteret laboratorium alt efter behov. Her skal det testes, hvordan udstyret fungerer i brug og hvorvidt der kan være utætheder eller andre sikkerhedsmæssige problemer i konstruktionen.

  • Både i Danmark og i vores nabolande er der registreret nogle uheld med campingudstyr med gasbrændere, og kontrolindsatsen kan give et større kendskab til, hvordan tilstanden er på området, siger kommunikationskonsulent Lone Hansen fra Sikkerhedsstyrelsen.

De produkter, der skal testes, vil som udgangspunkt blive udtaget i detailbutikker og internetbutikker i løbet af foråret og sommeren 2014.

Godkendelse og mærkning

Gasforbrugende apparater skal ifølge Gasreglementet være godkendte (DG- eller CE-mærkede) til installationsformålet, og de skal være godkendt til de gaskvaliteter og tilslutningstryk, der er tilladt i Danmark. Godkendelsen omfatter mærkning samt installations- og brugs­­anvisninger. Vejledningerne skal være på dansk ved markedsføring i Danmark.
