Børnefamilier laver mest ulovligt elarbejde

Pressemeddelelse
5. december 2003

Børnefamilier lever livet farligt. Det er de unge familier, der udfører mest gør-det-selv elarbejde og mest ulovligt elarbejde. Elektricitetsrådet har lavet en gratis guide, der viser, hvilket elarbejde man selv må lave, og hvordan man gør det sikkert.

Elektricitetsrådet har netop gennemført en undersøgelse af 1.500 parcelhusejeres gør-det-selv elarbejde. Undersøgelsen viser, at det er de unge familier, der udfører mest gør-det-selv elarbejde og mest ulovligt elarbejde. Den typiske gør-det-selv person er en mand mellem 30-39 med små børn i huset, siger Rikke Kristiansen, sekretariatschef i Elektricitetsrådet.

Hvert år opstår der omkring 3.300 elbrande i Danmark og statistikken viser, at hovedparten af brandene sker i private boliger. 8 ud af 10 boligejere lever med skjulte fejl i elinstallationerne. Fejl der i mange tilfælde er så alvorlige, at de kan starte brand. Og mange fejl skyldes forkert udført eller direkte ulovligt gør-det-selv arbejde på de elektriske installationer.

- Det er vigtigt, at man sætter sig ind i reglerne, før man kaster sig ud i gør-det-selv elarbejde, og overlader det farlige elarbejde til autoriserede elinstallatører. Over halvdelen af husejerne er i tvivl om reglerne for elarbejde. Og den anden halvdel, der mener at kende reglerne, er paradoksalt nok dem, der samtidig udfører mest ulovligt gør-det-selv elarbejde, siger Rikke Kristiansen.

Reglerne for gør-det-selv elarbejde er beskrevet i folderen "Gør det elsikkert", som kan bestilles gratis på Elektricitetsrådets hjemmeside www.elraadet.dk. Folderen giver også gode råd om, hvordan man sikkert laver elarbejde i hjemmet.
