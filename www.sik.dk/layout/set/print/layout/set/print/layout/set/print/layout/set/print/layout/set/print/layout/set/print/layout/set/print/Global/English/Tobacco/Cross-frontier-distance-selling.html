Cross frontier distance selling

If you want to market tobacco to consumers by cross frontier distance selling, you shall be registered with the Danish Safety Technology Authority.


Retailers shall be registered
All retailers, who want to market tobacco to consumers by cross frontier distance selling (as an example online trading), shall be registered with the Danish Safety Technology Authority.

This applies both if the retailer has established in Denmark and wants to sell to consumers in another EU/EEA country and if the retailer has established in another EU/EEA country and wants to sell to consumers in Denmark.

Moreover, retailers shall run an age control system to prevent sale of tobacco to persons under 18 years.

By cross frontier distance selling is meant distance selling to consumers where the consumer at the time of ordering the product is in another EU/EEA country than the country where the retailer has established.

If you want to market the product only within Denmark’s borders, it is not about cross frontier distance selling.

How to do
