Det går for langsomt med tredjepartskontrollen

Sikkerhedsstyrelsen har siden 2008 sendt breve ud til mere end 3.500 installatører med besked om at gennemføre en tredjepartskontrol af deres kvalitetsstyringssystem. Det er en stor og meget ressourcekrævende opgave, og styrelsen oplever desværre, at det stadig går for langsomt hos nogle installatører.

Eleftersynet tager form

Ny ordning: Folketinget har vedtaget en lov, der gør det obligatorisk at få lavet et eleftersyn, hvis man vil have en ejerskifteforsikring ved hushandel. Sikkerhedsstyrelsen er i fuld gang med at få alt på plads – og forventer, at installatørerne kan tilmelde sig fra april 2012.
