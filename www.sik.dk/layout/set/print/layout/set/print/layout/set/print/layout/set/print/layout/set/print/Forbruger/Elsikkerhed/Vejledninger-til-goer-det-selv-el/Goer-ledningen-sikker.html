Gør ledningen sikker

Læs om forskellige typer af ledninger, og hent gode råd om, hvordan du kan holde styr på dine ledninger. Læs også om ledninger i den faste elinstallation.


Ledninger til elapparater og forlængerledninger
Man må gerne selv skifte en ledning til et elapparat eller en forlængerledning. De typer ledninger, der bliver brugt mest, kan inddeles i tre grupper:

  1. Plastkappeledning
  2. Gummikappeledning
  3. Strygejernsledning

 Læs mere i vejledningen om ledninger:

  • Ledninger til elapparater, forlængerledninger og den fast elinstallation (PDF)

Hold styr på ledningerne
Hvis man holder styr på boligens ledninger, er der mindre sandsynlighed for brandfare. Derfor skal du undgå "fuglereder" af ledninger.

Genkend lederne på farverne
Du kan i visse tilfælde få stød eller lave en kortslutning, hvis du bytter rundt på lederne, fx når en stikprop skal monteres på en ledning.

Ledninger i den faste installation
Man må ikke selv udskifte en ledning eller installere nye i den faste elinstallation. Den faste elinstallation er en del af boligens elinstallation, som du ikke må tage med dig, hvis du flytter.

Det er fx målertavlen og fastmonterede stikkontakter.

Undersøg ledningerne, inden du går i gang
For at elinstallationen skal blive ved med at være sikker, bør du undersøge ledningerne for skader ved ethvert indgreb.
