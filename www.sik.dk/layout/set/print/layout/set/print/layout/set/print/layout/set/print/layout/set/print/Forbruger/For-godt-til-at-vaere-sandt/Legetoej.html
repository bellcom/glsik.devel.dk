Legetøj

Legetøj bliver farligt, når dit barn kan blive kvalt, rive sig eller få fingrene i klemme.

 

Kvælningsfare opstår, hvis dit barn kan tage smådele af legetøjet. Smådele kan sætte sig fast i halsrøret, hvis barnet tager dem i munden. Du kan teste for smådele med vores legetøjstester.

Læs mere om legetøjstesteren her

Lange snore er også en risiko, fordi de kan hænge fast i andre ting, når barnet leger, og de kan vikle sig rundt om barnets hals. Snore skal være kortere end 22 cm.

Hvis legetøjet har skarpe kanter er der risiko for at barnet kan rive sig.

Risikoen for at dit barn kan få fingrene i klemme opstår, hvis der bevægelige dele i legetøjet, som kan klappe sammen om barnets fingre.

Batterier kan også skabe farlige situationer. Du kan sikre dit barn ved at tjekke, om batteriklappen sidder ordenligt fast. Der er krav til, at man skal bruge skrutrækker eller andet værktøj til at fjerne batteriklappen på legetøj. Det er særligt små knapcellebatterier, der er farlige, fordi barnet nemt kan tage dem i munden og sluge dem.

Læs vores fire gode råd til, hvordan du kan handle mere sikkert på nettet.

  1. Ligger den butik du handler med uden for EU?
  2. Er der risiko for kortslutning, stød eller brand?
  3. Kan dit barn blive kvalt, rive sig eller få fingrene i klemme?
  4. Sælger du produkter videre, som er købt uden for EU? Så er det dit ansvar, hvis produktet er farligt.
