Pressemeddelelser 2013

Brandfare ved solcellepaneler fra europæisk producent

Sikkerhedsstyrelsen har modtaget en europæisk advarsel om brandfare ved et bestemt type solcelle-system fra Holland. Det drejer sig om fabrikatet Scheuten Solar system. Mærket er Multisol type P6-48, P6-54, P6-60 og P6-66. Styrelsen har nu haft kontakt med de danske importører af panelerne, som har taget kontakt til deres samarbejdspartnere, så solcellerne bliver fundet og udbedret.
