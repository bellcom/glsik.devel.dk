Fremstilling, import og salg

Se, hvordan du søger virksomhedsgodkendelse og læs reglerne for salg og opbevaring af fyrværkeri. Du kan også se en liste over alle godkendte fyrværkeriartikler.

Information til dig der sælger fyrværkeri

Vi har samlet information til dig, der sælger fyrværkeri i detailhandlen. Læs om, hvilke regler du skal overholde, link anmeldelsesblanket og se vejledning til dig der sælger fyrværkeri.

Salg og opbevaring af fyrværkeri

Se, hvilke tilladelser henholdsvis Sikkerhedsstyrelsen og kommunerne giver ved fyrværkeri. Hent også link til spørgsmål og svar om salg og opbevaring af fyrværkeri.

Virksomhedsgodkendelse af fyrværkerivirksomheder

Sikkerhedsstyrelsen administrerer virksomhedsgodkendelser på fyrværkeriområdet. Og vi varetager godkendelser til fremstilling og import.

Farlige fyrværkeriartikler

Sikkerhedsstyrelsen laver markedskontrol af nytårsfyrværkeri. Når vi finder en fyrværkeriartikel, som er farlig eller har en fejl, skal den trækkes tilbage fra markedet.

Sælg nytårsvarer sammen med fyrværkeri

Du kan søge dispensation til at sælge nytårsvarer sammen med fyrværkeri fra et salgssted i det fri.
