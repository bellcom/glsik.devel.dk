2003

Husk også elsikkerheden (Oktober 2003)

Ved by- og markedsfester etableres typisk en lang række midlertidige elektriske installationer. Ligesom ved faste elinstallationer er det vigtigt at tage højde for sikkerheden, så man undgå ulykker og brande forårsaget af elektricitet.

Ny standard for adaptere til industribrug (Marts 2003)

I Danmark er der flere afvigelser til den nye standard for adaptere til industribrug. Disse afvigelser skal kunne dokumenteres af fabrikanten eller importøren.

Få elsikkerheden i orden (Oktober 2003)

Sikkerhedsstyrelsen har ved stikprøvekontrol konstateret, at over halvdelen af alle forsamlingslokaler ikke har den krævede elsikkerhedsattest. Desuden blev der fundet mange fejl og mangler ved de elektriske installationer.

Elektricitetsrådet orienterer i nye rammer (December 2003)

Sikkerhedsstyrelsen overtager ved årsskiftet Elektricitetsrådets opgaver. Det betyder, at Elektricitetsrådet nu orienterer for sidste gang. Men der er stadig behov for information om direktiver, stærkstrømsbekendtgørelsens forskellige afsnit og de tilhørende standarder og normer. Derfor vil vi i Sikkerhedsstyrelsens regi fortsat skrive artikler til fagblade.

Krav til udligningsforbindelser

Siden 1994 har der været krav om forskellige udligningsforbindelser i nye bygninger.
