Transitional rules

Transitional rules shall ensure a smooth phase-in of the new rules.


To ensure a smooth phase-in of the new rules in the tobacco field, transitional rules have been implemented in the field.

Reference is made to section 15 “Ikrafttræden and overgangsbestemmelser m.v.” of Act on tobacco, etc.
