Sikkerhedsstyrelsen har givet forbud mod over 14.415 e-cigaretter og -væsker

Efter det første år med regler og kontrol af e-cigaretter og -væsker har Sikkerhedsstyrelsen gjort tallene op. Ved kontrol af 847 butikker og hjemmesider er der blevet udstedt forbud mod salg af mere end 14.415 produkter.


PRESSEMEDDELELSE: Den 7. juni 2016 kom der for første gang et regelsæt i forhold til e-cigaretter i Danmark. Reglerne trådte i første omgang i kraft for væsker med nikotin. Væskerne bliver til damp i e-cigaretterne og kan derefter inhaleres.

Siden har Sikkerhedsstyrelsen udført kontrol hos 847 danske og udenlandske hjemmesider og fysiske butikker i Danmark. Det er indtil videre blevet til 14.415 forbud mod salg af produkter, som ikke levede op til reglerne. Langt hovedparten af forbuddene handler om e-væsker.

Det meste af salget af e-cigaretter foregår på nettet. De fleste af de ulovlige produkter er også fundet på hjemmesider. Danske webshops tegner sig for 6.698 af forbuddene, mens udenlandske hjemmesider har fået 7.346 forbud mod salg af ulovlige produkter. Fysiske butikker har det første år fået 371 forbud.

- Tallene viser med al tydelighed, at især vores indsats på nettet har været vigtig. Det er her, det primære salg af e-cigaretter og –væsker foregår. Samtidig er det på hjemmesider, især udenlandske, hvor vi har fundet virkelig mange produkter, der strider mod reglerne, siger kontorchef i Sikkerhedsstyrelsen Jan Møller Mikkelsen.

Hvad angår de fysiske butikker, står de både for en mindre del af salget og overskridelse af reglerne. De opfølgende besøg, som Sikkerhedsstyrelsen har været på i butikkerne, har vist, at langt hovedparten heldigvis retter ind med det samme efter forbuddene mod salg.

- Vi glæder os over, når forhandlerne retter ind efter reglerne. I den kommende tid venter næste skridt i markedsovervågningen af e-cigaretter, når vi også begynder at køre kontrol på hardware-delen af produkterne, siger Jan Møller Mikkelsen.

For yderligere informationer kan pressekonsulent Lone Hansen kontaktes på telefonnummer 25 43 16 21.

Fakta

  • Den 7. juni 2016 trådte de første regler om e-cigaretter i kraft. De gælder alle e-cigaretter og genopfyldningsbeholdere med nikotin, der ikke er godkendt som lægemidler.
  • Efter den dato måtte der kun være produkter på hylderne, som levede op til reglerne, og produkterne skulle også være anmeldt til Sikkerhedsstyrelsen.
  • Ved 847 kontroller af fysiske butikker og hjemmesider i perioden 7. juni 2016 til 8. juni 2017 gav Sikkerhedsstyrelsen 14.415 forbud mod produkter, som enten ikke levede op til reglerne, ikke var anmeldt korrekt eller begge dele. Hovedparten af forbuddene var e-væsker og genopfyldningsbeholdere med nikotin.
  • Fra den 7. juni 2017 kom der også fuld regulering på hardware-delen. Det vil sige, at overgangsperioden for at sælge gamle produkter udløb.
