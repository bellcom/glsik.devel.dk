Spørgsmål og svar om elarbejde

Få svar på mange af de faglige spørgsmål indenfor elbranchen. Ved at klikke på spørgsmålene kommer svarene frem.

Nye regler om el er trådt i kraft den 1. juli 2017.

Læs mere om de nye regler på  www.elsikkerhedsregler.dk  

ATEX

ATEX står for "Atmosphere explosible". Det er fransk og betyder eksplosiv 'atmosfære'. Vi har samlet en række spørgsmål og svar inden for ATEX.

HFI- og HPFI-afbrydere

HFI- og HPFI-afbryderen - også kaldet fejlstrømsafbryder - er nødbremsen i de elektriske installationer. Vi har samlet en række spørgsmål og svar om HFI- og HPFI.

Eleftersyn

Eleftersyn er en del af huseftersynsordningen og obligatorisk, hvis der skal tegnes en ejerskifteforsikring. Læs mere eleftersynsordningen på Eleftersynsinfo.dk

Solceller

Flere og flere bruger grøn energi som en del af deres energikilde og får fx installeret solceller. Vi har samlet en række spørgsmål og svar om solcellesystemer.

Vi har samlet en række spørgsmål og svar om elprodukter.

Elinstallationer

Vi har samlet en række spørgsmål og svar om elinstallationer.

Kemi i elektronik

Kend din rolle og dit ansvar – og sæt dig ind i hvilke regler, der er for kemi i elektronik. Miljøstyrelsen har lavet en informationskampagne til virksomheder, og Sikkerhedsstyrelsen har bidraget med info om batterier.
