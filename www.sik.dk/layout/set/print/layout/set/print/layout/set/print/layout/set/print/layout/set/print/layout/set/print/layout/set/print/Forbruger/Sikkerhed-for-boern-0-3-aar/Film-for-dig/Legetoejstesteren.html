Lav din egen legetøjstester

Har du en legetøjstester? Nu kan du lave din helt egen print-selv legetøjstester.
Print-selv testeren er vejledende og giver dig en idé om, hvornår mindre dimser er så små, at de er farlige for småbørn i alderen 0-3 år. Hent print-selv testeren her:

Print-selv legetøjstester (PDF)

Se filmen og lær, hvordan du samler din egen legetøjstester.

 

 

Sådan tester du legetøjet

Sikkerhedsstyrelsens legetøjstester hjælper dig til at måle, om smådelene i og uden for hjemmet er så små, at de er farlige for dit barn i alderen 0-3 år.

 

  Sådan tester du de små dele

  1. Put legetøjet eller andre små dimser i testeren.
  2. Kan dimsen ligge i testeren uden at rage op over kanten, er den farlig for dit barn.
  3. Du må ikke tvinge noget ned i testeren. Uanset, hvad du vil teste, så skal det altid selv kunne glide ned i testeren og ligge der uden tvang.

Legetøjstesteren svarer til en barnehals, og måler 3,17 cm i diameteren, 2,54 cm på den korte side og 5,71 cm på den lange side.
