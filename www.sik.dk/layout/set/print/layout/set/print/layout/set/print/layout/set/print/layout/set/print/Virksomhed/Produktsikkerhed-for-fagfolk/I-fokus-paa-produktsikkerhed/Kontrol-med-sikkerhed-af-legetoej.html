Kontrol med sikkerhed af legetøj

Sikkerhedsstyrelsen gennemfører i 2009 en særlig kontrol af sikkerhed og mærkning af legetøj, som sælges i butikker med blandet varesortiment. Det vil sige i butikker, hvor legetøj ikke er den primære varegruppe.


Fokus vil være på legetøj, som vi vurderer, er til børn under 3 år. Dette afgør vi blandt andet ud fra udformning, farve mv. som angivet i en international Guideline for området. Mærkningen af legetøjet vil ikke være afgørende for vores vurdering.

Krav om dokumentation
Styrelsen vil udtage legetøj til alle aldersgrupper, hvis der er oplagte sikkerheds- eller mærkningsproblemer. Importør/producent vil bl.a. blive bedt om at fremlægge teknisk dokumentation for legetøjets sikkerhed.

Legetøjet bliver testet
Sikkerheden bliver testet på et laboratorium. Viser legetøjet sig at være farligt, eller lever produktet ikke op til standard eller typegodkendelse, vil importør/producent skulle betale regningen for testen. Resultatet af kontrollen vil være klar i løbet af efteråret.

Regler for legetøj
Legetøj er defineret som produkter, der er konstrueret eller bestemt til legeformål for børn under 14 år. Legetøjsreglerne er fælles for hele EU, og trådte i kraft i Danmark i 1991. Reglerne bygger på tre vigtige elementer:

  • Legetøjet skal opfylde sikkerhedskravene i Legetøjsbekendtgørelsen
  • Importøren og/eller producenten skal kunne dokumentere, at legetøjet opfylder disse krav
  • Legetøjet skal være mærket med et CE-mærke, importørens navn og adresse og eventuelle advarsler.

Baggrunden for sikkerhedskravene er viden om børns adfærd, deres mål og vægt på forskellige udviklingstrin, erfaring og oplysninger fra virkelighedens verden om ulykker, som børn har været ude for i forbindelse med leg og legetøj.

Du kan læse mere om resultaterne af overvågningsindsatsen ved at følge linket til rapporten.
