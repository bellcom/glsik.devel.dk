Laserpointere fra Olsens-it

Klik på billedet:

Salgsstop og tilbagekaldelse fra forbrugere:

  • 30 milliwatt grøn laserpen, Reference: 10027
  • Trefarvet laserpegepind (grøn, rød, blå) m. – 100mW, Reference: 20586

Du skal stoppe med at bruge laserpointeren og holde den væk fra børn. Øjets nethinde kan få vedvarende skader selv ved kortvarig belysning med laserlys i klasse 3B, der overstiger 5 mW. I værste tilfælde kan der ske permanent tab af synet på det udsatte øje.

Følgende produkter må ikke længere sælges:

  • 5 mw blå laser + LED m. nøglering, Reference: 10029
  • 5 milliwatt blå laserpen, Reference: 10028
  • Rød laser pointer Bjælkekuglepen - 5mW, Reference: yr63
