Gode råd om el

Elinstallationerne i dit hus skal være sikre. Hent gode råd om, hvordan du kan tjekke op på, om boligens elinstallationer fungerer, som de skal.

Træk stikket når det tordner

Når uvejret raser, og lyn og torden farer hen over himlen, bliver mange mennesker nervøse.

Brug opladeren sikkert

Når du lader elektrisk udstyr op, så tænk over, at el kan udgøre en brandfare, ligesom der kan være risiko for elektrisk stød.

Sikkerhedsstyrelsen går på jagt efter farlige opladere

Sikkerhedsstyrelsen gennemfører en kontrolkampagne af opladere i Danmark.

Sådan undgår du stød, når du gør haven forårsklar

Elektriske hækkeklippere, kantklippere og græstrimmere har gjort mange haveejeres liv en hel del nemmere. Af hensyn til sikkerheden er der en række ting, du bør være opmærksom på, før du kaster dig ud i arbejdet med de eldrevne redskaber.

Hvorfor skal du teste din fejlstrømsafbryder?

Fejlstrømsafbryderen er nødbremsen i dit hjem. Den sidder i dit elskab og beskytter dig mod livsfarligt stød ved at afbryde strømmen, hvis der opstår en fejl i et elapparat. En fejlstrømsafbryder kan redde liv.

Undgå ulykker ved komfuret

Hvis en gryde eller pande bliver glemt på komfuret, eller der bliver lagt en avis eller andet brændbart på en tændt kogeplade, kan det starte en komfurbrand. Hvert år opstår der mere end 500 komfurbrande på grund af forkert eller uforsigtig brug af komfurer. Komfurbrande er i gennemsnit årsag til fem dødsfald årligt.

Gode råd om el før du handler hus

En gennemsnitlig dansk bolig får skiftet tapet 17 gange, mens den kun får skiftet elinstallationer én gang. Derfor er det vigtigt, at man får tjekket elinstallationerne, inden man køber hus.

Sikker el i sommerhuset

Få en el-sikker start på sæsonen i dit sommerhus. Hent råd om elsikkerhed, når du åbner eller lukker sommerhuset.

Få styr på solcellerne

Tjek sikkerheden på dit solcelleanlæg. Der kan være en sikkerhedsrisiko ved nogle solcelleinstallationer, fordi der ikke er installeret den type fejlstrømsafbryder, som kræves til inverteren.

Bevar jordforbindelsen i dine hårde hvidevarer

I 2011 blev stikkontakter med sidejord lovlige i Danmark. Det betyder, at der nu er tre forskellige stikkontaktsystemer, som en autoriseret elinstallatørvirksomhed må sætte op i danske hjem.
