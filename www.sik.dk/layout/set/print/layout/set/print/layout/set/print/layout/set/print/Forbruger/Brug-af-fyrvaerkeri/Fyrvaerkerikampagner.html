Fyrværkerikampagner

VR-film, gode råd, YouTube, Facebook. Sikkerhedsstyrelsen sætter alle sejl til for at oplyse danskerne om, hvordan man mest sikkert anvender fyrværkeri. Læs og se de forskellige kampagne-elementer her.

Kampagne - oplev en fyrværkeriulykke i virtual reality

I biografer, busser og tog, på Youtube og i virtual reality vil Sikkerhedsstyrelsen op til årsskiftet 2017-2018 lade dig opleve en fyrværkeriulykke, som om du selv er offer for den.

19.000 skoleelever har lavet fyrværkerifilm

Mere end 19.000 skoleelever har i november og december 2017 arbejdet med fyrværkeriråd i undervisningen og lavet kampagnefilm med fyrværkeriråd.

To Youtubere fyrer den af

Alexander Husum og GEX fyrer den af for at give flest mulige et sikkert nytår – YouTuber sprænger sig selv i luften i ny YouTube-video.

Raketfilm

Op til nytåret 2014, 2015 og 2016 viste Sikkerhedsstyrelsen en 3D-reklamefilm i biografer landet over. Filmens formål var at få folk (især yngre mænd) til at huske at bruge sikkerhedsbriller og holde afstand, når de fyrer nytårskrudt af. Filmen blev også vist i mange af landets tog og busser op mod nytår. Se filmen her.

Ulykker med ulovligt krudt rammer ikke kun dig selv

Sikkerhedsstyrelsen stod i 2015 og 2016 bag en kampagne mod ulovligt fyrværkeri i samarbejde med SKAT og politiet.
