Elektronisk kommunikation

Se, hvordan du sender sikker e-post til Sikkerhedsstyrelsen, og find de nødvendige oplysninger for at sende en elektronisk faktura til os.


Sikker og krypteret e-post

Vi er parat til at modtage krypteret e-post på vores almindelige adresse: sik@sik.dk .
Sikker e-post kan kun sendes til denne adresse.

For at du kan sende krypteret e-post til Sikkerhedsstyrelsen, skal du have et særligt certifikat, som er særligt knyttet til vores e-post-adresse.

Certifikatet findes hos NemID, hvor du skal søge efter sik@sik.dk .

Hent certifikat hos NemID

Hvis du har brug for teknisk support i forbindelse med installation og brug af certifikater og krypteret post, kan du få hjælp hos NemID.

Læs om krypteret post hos NemID

Elektronisk fakturering

Fakturaer til Sikkerhedsstyrelsen kan kun modtages elektronisk på EAN-nummer 5798000028005 .

Ud over EAN-nummeret skal du påføre fakturaen en personreference (dvs. hvem i Sikkerhedsstyrelsen, der skal have fakturaen).

Brug dette link til at taste elektronisk faktura

Overgang til elektronisk fakturering

Overgangen til elektronisk fakturering er et led i effektiviseringen af den offentlige sektor og gælder alle offentlige myndigheder i staten og kommunerne, som både skal kunne sende og modtage regninger elektronisk.

Digital post fra Sikkerhedsstyrelsen

Virksomheder har siden 1. november 2013 haft pligt til at kunne modtage digital post fra det offentlige, og borgere over 15 år har haft samme forpligtelse siden 1. november 2014.

Sikkerhedsstyrelsen sender digital post, og det er modtagerens ansvar at have adgang til de breve, der modtages digitalt.

På www.virk.dk  og www.e-boks.dk  er der information om, hvordan virksomhederne får en digital postkasse. Privat personer kan se hvordan de opretter en digital postkasse på www.borger.dk  og www.e-boks.dk . 

På virk.dk og borger.dk er der også oplysninger om, hvor man kan få hjælp til brugen af den digitale postkasse.
