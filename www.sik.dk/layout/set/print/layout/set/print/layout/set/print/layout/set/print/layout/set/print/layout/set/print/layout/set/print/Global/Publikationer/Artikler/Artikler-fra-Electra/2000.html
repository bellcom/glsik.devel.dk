2000

Når målerskabet brænder (Februar 2000)

Elektricitetsrådet konstaterede i 1998 og 1999 et større antal brandtilløb og enkelte brande i udvendige målerskabe for stikbensmålere.

Installation af fjernvarmemålere (December 2000)

Der er i flere artikler i pressen i efteråret 2000 stillet spørgsmål ved den elektriske sikkerhed i forbindelse med installation af varmemålere i Århus. Det drejer sig om installationer, der er udført med en separat transformer anbragt i gruppetavlen efter anvisningerne i ELRÅD-MEDDELELSE, Installationer nr. 5/98.

Beskyttelse mod jævnfejlstrømme må vælges med omhu (Februar 2000)

Anvendelse af moderne elektronik i elektrisk materiel især i forbindelse med frekvensomformere medfører, at der kan optræde fejlstrømme med et højt indhold af jævnstrøm. Det må der tages hensyn til ved valg af midler til beskyttelse mod indirekte berøring.

Tavleulykker rammer installatørernes ansatte (Januar 2000)

Antallet af tavleulykker er fortsat højt. Elektricitetsrådet har analyseret 131 tavleulykker, der er sket indenfor en femårs periode fra 1994 - 1999 og set nærmere på, hvorfor ulykkerne opstår.

Elinstallationer på diskoteker skal kontrolleres (Juli 2000)

Ved eftersyn på diskoteker o.l. i 1999 blev der konstateret adskillige fejl og mangler ved de elektriske installationer, ligesom nød- og panikbelysningen i mange tilfælde slet ikke kunne fungere.
