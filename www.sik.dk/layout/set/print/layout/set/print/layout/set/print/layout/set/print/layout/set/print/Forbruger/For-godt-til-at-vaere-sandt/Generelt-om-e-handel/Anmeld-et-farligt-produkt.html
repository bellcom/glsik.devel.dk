Anmeld et farligt produkt

Hvis du oplever, at et produkt er farligt, kan du altid henvende dig til os.


Du kan kontakte os på flere måder:

  • Skriv en mail til sik@sik.dk
  • Ring til os på tlf.: 33 73 20 00
  • Send os en messenger-besked

Det er vigtigt, at du giver os så mange informationer som muligt om produktet, så vi hurtigst muligt kan gå videre med sagen. Vi har brug for at få:

  • Dine informationer: navn, tlf. nr. og e-mailadresse, så vi kan kontakte dig, hvis vi brug for flere informationer.
  • Beskrivelse af, hvordan produktet er farligt
  • Billeder af produktet
  • Information om, hvor produktet er købt

Vi tager alle henvendelser seriøst, så tøv ikke med at kontakte os

Hvornår er et produktet farligt?
Legetøj:

  • Hvis smådele kan falde af
  • Hvis der er lange snore
  • Hvis man kan åbne batteriklappen uden en skruetrækker eller andet værktøj

Elektronik:

  • Hvis du har adgang til spændingsførende dele
  • Hvis produktet overopheder eller eksploderer
