Som autoriseret virksomhed kan du logge ind i følgende fagsystemer

Eleftersynsordningen

Rollen som virksomhed:
Som virksomhed har man adgang til at se, men ikke til at oprette rapporter. I denne rolle kan man oprette administratorer og medarbejdere. Hvis virksomheden har flere afdelinger, kan det være en fordel at have flere administratorer. Som virksomhed kan stamdata, kortoplysninger og markedsføringsoplysninger redigeres.  

Brugernavn: Virksomhedens cvr-nummer.
Password: Se i virksomhedens digitale postkasse (bliver afsendt ved tilmelding)

Link til log ind som virksomhed i eleftersynsordningen

Rollen som administrator:
Som administrator har man adgang til at oprette, godkende og afslutte rapporter. Her har man mulighed for at oprette medarbejdere under sig. I denne rolle kan man se rapporter, som er oprettet af administratoren og af de medarbejdere, der er under denne.

Brugernavn: Administrators mailadresse
Password: Se i administrators mail (bliver afsendt ved oprettelse/aktivering af administrator)

Link til log ind som administrator i eleftersynsordningen

Rollen som medarbejder:
Som medarbejder defineres adgangen til at påbegynde, gennemføre, godkende og afslutte rapporter af administratoren. I rollen som medarbejder har man kun adgang til at se rapporter, som medarbejderen selv har oprettet.

Brugernavn: medarbejders mailadresse
Password: se i medarbejders mail (bliver afsendt ved oprettelse/aktivering af medarbejder)

Link til log ind som medarbejder i eleftersynsordningen

Kontrolordningen for forsamlingslokaler

Elektriske installationer i forsamlingslokaler skal efterses hvert andet år af en autoriseret elinstallatørvirksomhed. Læs mere om, hvad kontrolbesøget omfatter, og hvordan det bliver indrapporteret til Sikkerhedsstyrelsen.

Link til indrapportering af kontrol

Virksomhedsprofil

Virksomhedsprofilen er blevet nedlagt.

Den 1. november 2018 går vi i luften med et helt nyt sik.dk. I den forbindelse har vi valgt at nedlægge virksomhedsprofilen. Det betyder, at du ikke længere kan logge ind og tilføje ekstra kontaktoplysninger til din virksomhed. Autorisationsregisteret er helt centralt for os, og virksomheder og forbrugere vil derfor stadig kunne søge efter autoriserede virksomheder ud fra navn, adresse, CVR-nummer, autorisationsnummer og autorisationsområde.

Installationsblanket

I en overgangsperiode skal man logge ind på ”Installationsblanketten” hos Dansk Energi og Dong Energy med det gamle autorisationsnummer for den tidligere ”Personlig ansvarlig”. Man kan derfor ikke bruge virksomhedens nye autorisationsnummer til at logge ind med.

Dansk Energi og Dong Energy oplyser, når overgangsperioden er slut.

Link til log ind side til Installationsblanketten.dk

Har du problemer med at logge ind, skal du kontakte Dansk Energi eller Dong Energy.
