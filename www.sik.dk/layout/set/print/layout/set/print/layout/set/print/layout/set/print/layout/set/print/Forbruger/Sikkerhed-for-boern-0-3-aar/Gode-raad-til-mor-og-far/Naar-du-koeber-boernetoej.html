Når du køber børnetøj

Nyt tøj til dit barn skal være uden lange snore eller smådimser, som børn risikerer at blive kvalt i. Få 6 sikre råd om børnetøj.


Når du er på indkøb og skal købe nyt tøj til dit barn, er det en god idé at have særligt øje på, om der er snore eller smådimser, som kan udgøre en fare for dit barn. 

6 gode råd om sikkert børnetøj
  1. Tøj til børn op til syv år (højde 1,34 m) må ikke have bindebånd eller snore i hoved- og halsområdet. Tøj til børn mellem 7 og 14 år må ikke have bindebånd i hoved- og halsområdet, der er længere end 75 mm eller snore med frithængende ender.
     
  2. Forsikr dig, at smådelene på dit barns tøj sidder godt fast. Pailletter, knapper og perler er flotte – men også små nok til at ende i dit barns mund. 
     
  3. Kan tøjets smådele bestå legetøjstesteren ? Brug legetøjstesteren til at måle, om smådelene på dit barns tøj er for små.
     
  4. Giv ikke dit barn halstørklæde på, når det leger. Pyntehalstørklæder er smukke. Men dit barn kan blive kvalt, hvis det bliver viklet ind i grene eller sætter sig fast i en dør. 
     
  5. Tjek, at tøjet passer til dit barns alder. Følg altid anbefalinger om alder og advarsler, og tag det alvorligt. Særligt i forhold til børn under 3 år – øjemål og sund fornuft er ikke nok!
     
  6. Køb kun legetøj, som er CE-mærket . Fabrikanten angiver med CE-mærket, at produktet lever op til sundheds- og sikkerhedskravene i EU. 

Reglerne om børnetøj hører ind under Produktsikkerhedsloven .
