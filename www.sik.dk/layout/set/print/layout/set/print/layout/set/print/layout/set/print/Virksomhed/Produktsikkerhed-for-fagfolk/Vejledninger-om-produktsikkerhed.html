Vejledninger om produktsikkerhed

Vi laver vejledninger omkring produktsikkerhed. Se bl.a. hvad du skal være opmærksom på som iværksætter, distributør af legetøj eller når du indkøber elprodukter.

Standarder inden for elektroteknik og forbrugerprodukter

På Dansk Standards hjemmeside kan du søge blandt danske, europæiske og internationale standarder inden for fx elektroteknik og forbrugerprodukter.

Vejledning for distributører af legetøj

Du er distributør, når du sælger legetøj, som du har købt af en fabrikant eller importør inden for EU. Hvis du vælger at sætte dit navn på legetøjet, skifter du status til fabrikant.

Sikre trampoliner til privat brug

Trampoliner til privat brug er legetøj og skal overholde reglerne om legetøjssikkerhed. Der må således ikke være nogen uacceptabel fare for sikkerheden, hvis trampolinen bruges, som man må forvente, at børn vil anvende den.

Sælg kun sikre elprodukter

Uanset om du er importør, producent eller distributør, så har du altid et ansvar for, at dit produkt er sikkert. Hav derfor styr på den korrekte dokumentation for produktets sikkerhed, så du ikke importerer og sælger farlige produkter til dine kunder.

Vejledning for legetøj

EU Kommissionen laver løbende vejledninger til hjælp for fortolkningen af legetøjsdirektivet. Vi bruger typisk vejledningerne som grundlag for vores markedsovervågning.

Indkøb af elprodukter - en guide i elsikkerhed

Sikkerhedsstyrelsens guide i elsikkerhed til importører og fabrikanter er ment som inspiration, og kan lede dig på rette vej, inden du indkøber elapparater.
