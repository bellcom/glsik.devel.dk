Sikker påske

Påsken er en hyggelig tid, og måske kommer påskeharen også på besøg hos jer med små, lækre påskeæg og andre godter. Men pas på med påskeæg, småt legetøj og andet, der kan puttes i munden.


Det føles næsten ligesom at pakke gaver op til jul – spændingen ved at åbne et påskeæg, hvor der gemmer sig små godter og legetøj. Spændende er det, men de små ting, der gemmer sig i påskeægget, kan sætte sig fast i luftvejene, og specielt de runde og ovale former skal du være opmærksom på.

De større runde former kan sætte sig fast bagest i munden og blokere for luftpassage. De er ikke til at få ud uden at skade barnet. Smådelene blokerer først nede i halsen. Og risikoen for, at legetøj eller andre smådele ender i dit barns mund eller hals øges, når legetøj kombineres med noget, der kan spises. 

Det kan du gøre
  1. Adskil legetøj og mad/slik.
     
  2. Lav dit eget påskeæg med godter i stedet for at købe et færdiglavet. På den måde kan du sikre dig, at legetøj og slik ikke er blandet og at godterne passer til dit barns alder.
     
  3. Passer påskelegetøjet til dit barns alder? Følg altid anbefalinger om alder og advarsler og tag det alvorligt. Særligt i forhold til børn under 3 år – øjemål og sund fornuft er ikke nok!
     
  4. Giv ikke børn under 3 år slik eller legetøj med runde og ovale former.
     
  5. Kan påskeæggets smådele bestå legetøjstesteren? Brug Sikkerhedsstyrelsens legetøjstester til at måle, om smådelene i dit barns påskeæg er for små. Find den her
     
  6. Kender du CE-mærket? Fabrikanten angiver med CE-mærket, at produktet lever op til sundheds- og sikkerhedskravene i EU. Køb derfor altid kun legetøj til dit barn, som er CE-mærket.
