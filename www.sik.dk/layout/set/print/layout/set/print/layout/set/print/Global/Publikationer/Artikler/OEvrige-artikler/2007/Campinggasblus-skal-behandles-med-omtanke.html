Campinggasblus skal behandles med omtanke

Når du skal bruge flaskegas, skal du udvise omtanke. Gas på flaske eller dåse er en praktisk energikilde f.eks. i sommerhus, båd eller campingvogn.


Det er Sikkerhedsstyrelsens holdning, at følger du brugervejledningen, som skal følge med danske gasprodukter, kan det stort set ikke gå galt.

Styrelsen har intet kendskab til usikre gasprodukter på det danske marked, som fx kan eksplodere af sig selv. Når uheldet sker, er det stort set altid på grund af fejl i brugen af produktet. En typisk fejl sker i forbindelse med samling af gasblusset, hvor man f.eks. tvinger dåsen op i brænderen med risiko for, at gassen kan komme til at sive ud. Kommer dén gas i kontakt med ild, kan det gå rigtig galt. Følg derfor altid instruktionen vist på dåsen.

Følger du disse enkle råd, kan du være på den sikre side:

  • Læs og følg altid brugervejledningen
  • Pas på ved udsivende gas. Hører du en sivende lyd fra flasken, skal du holde den langt væk fra både ild og gnister.
