Indberetning af elulykker

Den 1. juli 2017 gælder de nye regler om drift af elektriske anlæg. Hvis der sker en ulykker af elektrisk karakter på et anlæg, skal du indberette det til os.


Du kan læse mere om indberetning af elulykker på elektriske anlæg på elsikkerhedsregler.dk .
