Vigtig sikkerhedsinformation: FROSTFRI Køleskab og FROSTFRI fryser.
Betjeningspanelet bag døren kan gå løs og dermed give adgang til strømførende dele.
Der er tale om produkter produceret i perioden uge 45/2015 til uge 7/2016.
Information om, hvornår køleskabet eller fryseren er produceret, fremgår af datostemplet, som kan findes ved at fjerne den nederste skuffe.
Produkter med datostempler fra 1545 til 1607 er omfattet af denne sikkerhedsinformation.
