Autorisationsregister

I Sikkerhedsstyrelsens autorisationsregister kan man søge eller tjekke, om en virksomhed har autorisation inden for el, vvs, kloak og gas.

Søg eller tjek din håndværker

Søg en autoriseret virksomhed. Selvom du søger inden for din egen kommune, vil resultatet også indeholde virksomheder fra andre kommuner. Det er fordi, listen viser alle de virksomheder, der arbejder i din kommune. Du kan sortere søgeresultatet på postnummer. Ud for hver virksomhed kan man se, hvilke arbejdsområder virksomheden har autorisation til.

Alle virksomheder, som har autorisation fra Sikkerhedsstyrelsen, kan søges frem i autorisationsregisteret.

Du har mulighed for at sortere på kommune og autorisationsområde. Du har også mulighed for at se telefonnummer, e-mail og hjemmeside. Det er oplysninger, som virksomheden selv indtaster. Har virksomheden ikke angivet noget, fremgår de kun med adresse og autorisationsoplysninger.
