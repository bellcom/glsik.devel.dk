USBECK type 1021, 47-58 mbar

Klik på billedet:

Bunsenbrænder USBECK type 1021, 47-58 mbar må ikke bruges, da den ikke lever op til de gældende sikkerhedskrav.
Tilslutningstrykket 47-58 mbar er beregnet til tyske installationsforhold, og slangespidsen er ikke den godkendte til danske installationsforhold efter dansk standard - DS 24.
Den påsatte slange kan derfor falde af under brug med gasudslip og ulykker til følge.
Produkttype: Bunsenbrænder
Produktnavn: USBECK type 1021, 47-58 mbar
Importør: TH Geyer Danmark ApS Byleddet 7 4000 Roskilde
Forhandler: Frederiksen Scientific A/S Viaduktvej 35 DK-6870 Ølgod
Anden information:

TH Geyer Danmark ApS, Byleddet 7, 4000 Roskilde har fået påbud om tilbagekaldelse af bunsenbrænder USBECK ,type 1021, 47-58 mbar.
Hvis du har købt denne bunsenbrænder USBECK type 1021, 47-58 mbar, anbefales du at frakoble bunsenbrænderen og rette henvendelse til fabrikanten eller leverandøren.
Dato: 15. juli 2016
